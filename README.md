# Anime Data Analysis using Python

## Description
This project performs basic data analysis and feature extraction
on an anime dataset using Python.

## Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Dataset
Anime dataset in CSV format containing information such as
anime name, genre, rating, episodes, etc.

## What I Did
- Loaded and explored the dataset using Pandas
- Cleaned missing and duplicate values
- Performed feature extraction
- Analyzed basic statistics of the dataset

## How to Run
1. Clone the repository
2. Install dependencies
3. Open the Jupyter Notebook
